# Belajar Data Science
repo gabut buat belajar data science walaupun nggak sampai expert karena pengen tahu doang.

### setup
```bash
    # clone repo ini
    git clone https://github.com/Zyprush18/data-science.git

    # masuk ke dalam directory
    cd data-science

    # buat virtual environtment python
    python -m venv <nama_env>

    # aktifkan environment
    # di linux atau mac os
    source <nama_env>/bin/activate
    # di windows
    <nama_env>/bin/activate

    # install dependency yang di butuhkan
    pip install -r requirements.txt

```
